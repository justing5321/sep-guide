Welcome to the 11th grade SEP class. Here you will find lots of work done by students in various topics.

Github:

Is a website where people can code and be able to share their projects with everyone wanting to further their coding knowledge.

Check out Anthony's project!

HTML:

Allows coders to have content in their project and structure their project the way you want to.

CSS+Bootstrap:

This subject is where the coder is the artist. They use css and bootstrap to style their website.

Js+jQuery:

The purpose of jQuery is to make it much easier to use JavaScript on your website. jQuery takes a lot of common tasks that require many lines of JavaScript code to accomplish, and wraps them into methods that you can call with a single line of code.
